# ⚽ لعبة "اعرف اللاعب" - Football Quiz Game

🎮 لعبة تفاعلية لاختبار معرفتك بالاعبي كرة القدم حول العالم

![لقطة شاشة اللعبة](screenshot.png)

## ✨ المميزات

- 🏆 أكثر من 50 لاعب من مختلف العصور والمنتخبات
- 🔥 4 مستويات صعوبة (سهل، متوسط، صعب، خبير)
- 💰 نظام كاش ونقاط متطور
- ⚡ 5 أنواع من Power-ups (مساعدة في اللعبة)
- 📊 نظام مستويات وخبرة (XP)
- 🏅 إنجازات وجوائز
- 🌙 واجهة عربية جميلة
- 📱 متجاوب مع الجوال

## 🎯 كيفية اللعب

1. اختر مستوى الصعوبة
2. اقرأ المعلومات عن اللاعب
3. اختر اللاعب الصحيح من 4 خيارات
4. استخدم Power-ups للمساعدة
5. اجمع النقاط والكاش وارفع مستواك

## 🚀 التقنيات المستخدمة

- HTML5
- CSS3 (مع متغيرات CSS والأنيميشن)
- JavaScript (ES6+)
- JSON لقاعدة البيانات
- LocalStorage للحفظ
- Web Audio API للأصوات
